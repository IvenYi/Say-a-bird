# 「说只鸟（Say a Bird）」UI/UX 美术风格规范

**版本**: V2 最终确认版

**定位**: 核心视觉资产与 UI 开发指南

### 整体视觉定位

**核心关键词**：治愈肌理、物种严谨、克制高级、呼吸感

本产品的视觉核心在于“温度与克制的平衡”。为了契合当代年轻人的“反内耗”诉求，同时满足观鸟群体对物种特征的执念，整体视觉抛弃了传统的黑色线条描边与低幼的 Q 版变形。天空背景如同高级的独立游戏般干净整洁，而飞入天空的情绪鸟类则带有绘本般的粉彩温度。

---

### 1. 天空画布的美术风格（背景与基底）

**整体原则**：无描边几何扁平风（Stroke-less Geometric Flat），极致干净，为带有肌理的鸟类提供清晰的展示舞台。

**背景设计**：

- **风格**：纯色块拼接与柔和渐变，拒绝任何具象的写实云朵或繁复细节。
- **色彩饱和度**：中低饱和度（40-60%），确保背景不会喧宾夺主。

**四个时段的色彩方案**：

- **清晨（6-8点）**：橙粉 → 浅蓝渐变，色温偏暖，唤醒感。
- **白天（8-17点）**：天蓝 → 白色渐变，明度最高，给人开阔与透气感。
- **黄昏（17-19点）**：暖金 → 灰紫渐变，最有情绪感染力和治愈感的时段。
- **夜晚（19-6点）**：深蓝 → 紫黑渐变，低明度但不压抑，保留极简的几何星光点缀。

**三栖基底（树木、水面、滩涂）**：

- **风格**：极简几何色块拼接（参考确定的树木 Moodboard）。无描边，依靠相邻色块的明暗对比区分树冠层次和水面波纹。

---

### 2. 鸟类插画的风格（核心视觉资产）

**确立风格：无描边、带噪点肌理的粉彩绘本风（Textured Pastel Flat Illustration）**

**具体特征**：

- **边缘与填充**：绝对无描边（No Strokes）。色块边缘带有粉彩、蜡笔或高级绘本特有的“粗糙噪点肌理（Noise Texture）”，带来强烈的手工温度和抚慰感。
- **形态与比例**：**拒绝过度低幼的 Q 版变形（如 1:1.5 头身比）**。严格保留该鸟类的核心生物学特征（如鹭的修长体态、长尾山雀的精准羽色分布、翠鸟的喙型），但在整体线条上做适度圆润化处理。
- **色彩**：提取真实鸟类羽色，但统一降低饱和度（30-50%），融入整体的治愈色调。
- **眼睛**：简化为干净的圆点或带有极小高光的纯色块，透出清醒或呆萌的“人设感”。

**为什么选择这种风格？**

1. **极强的情绪安抚力**：毛茸茸的粉彩噪点边缘比冷硬的线条更具亲和力，能瞬间降低用户的心理防线。
2. **获取观鸟圈认同**：精准的特征保留能让种子用户（观鸟爱好者）感受到产品的专业与用心。
3. **视觉层级分明**：干净几何的背景 + 带噪点肌理的鸟类，构成了完美的前后景反差，即使同屏 20 只鸟也不会产生视觉噪音。

### 3. UI 界面风格

**整体原则**：去边框化，依靠色彩区隔层级，留白即是 UI。

**配色方案**：

- **主色**：治愈系低饱和色（如灰豆绿 #A8D5BA 或 莫兰迪蓝 #87A9BE）。
- **文字色**：暖深灰（#4A4A4A）而非纯黑，极大减少阅读内耗与视觉疲劳。

**控件设计**：

- **无硬边框**：所有卡片、按钮均不使用实线描边。
- **卡片设计**：纯白或极浅灰底色，配以超大圆角（16px-24px）和极其微弱的弥散阴影（Blur 20px, Alpha 8%）。
- **主按钮**：“说只鸟”输入按钮采用大圆角药丸状，填充柔和的主色。

---

### 4. 动画与交互

**动画原则**：模拟自然物理的缓动，安静且克制。

- **鸟类飞入动画**：0.8 秒-1.2 秒。淡入（Opacity 0→1）配合轻微的贝塞尔曲线抛物线轨迹落入对应生态位。缓动函数使用 `ease-out`（先快后慢，宛如鸟类收翅降落）。
- **解锁新鸟特效**：避免刺眼的光效。使用柔和的同色系光晕在鸟类背后缓慢放大并淡出（Scale 1→1.5, Opacity 0.8→0），用时 1.5 秒。
- **页面切换**：所有层级切换均使用 0.3 秒的淡入淡出（Crossfade），摒弃自带侵略性的左右滑动推窗。

---

### 5. 分享图设计（社交传播核心）

**设计理念**：一张不暴露隐私的高级情绪海报。

**排版结构**：

- **主视觉区**：占据画面的 70-80%，完整截取带有当前时间段光照的几何天空与用户的粉彩鸟类。
- **词云融合设计**：**摒弃生硬散落的排版**。将提取出的无害情绪词云，使用手写体（如站酷快乐体），以上下错落的形式，**半掩半露地穿插、依附在画面底部的几何树冠或云层边缘**，仿佛是自然环境的一部分。
- **时间标记**：顶部或左上角，极简排版：“我的情绪天空 · 第 X 天”。
- **底部留白**：放置产品 Logo 与小程序码。

---

### 6. 最容易踩的坑（防坑指南）

❌ **强行 Q 版化 / 变成“球”**：为了可爱而把所有鸟都画成胖乎乎的圆球，这会彻底得罪观鸟群体，也会让 15 种鸟在小屏幕上失去辨识度。

✅ **尊重结构**：用几何块概括原本的体态，保留它们或修长、或威武的原本面貌。

❌ **画面中出现哪怕一根实线**：UI 界面和插画中混入传统的描边线条。

✅ **极致无描边**：死磕“Stroke-less”，只用色块的相接来交代结构，这是保持高级感的底线。

❌ **过度丰富的渐变与高光**：在鸟的身上画出立体的体积感和高光。

### 7. 参考锚点

- **插画质感**：Jon Klassen 的儿童绘本作品（如《我要把我的帽子找回来》），感受那种带有噪点肌理、情绪克制但极具张力的色块表达。
- **背景与 UI 融合**：《Monument Valley（纪念碑谷）》的几何色彩美学。
