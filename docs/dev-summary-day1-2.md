## 🔧 技术债务与注意事项

### 需要注意的问题

1. **语音输入功能**
   - 当前 input 页面中的语音输入功能需要接入微信语音识别插件
   - 建议先跳过，MVP 阶段可以只支持文字输入

2. **图片路径问题**
   - 所有页面中的图片路径都是占位路径
   - 素材准备好后需要统一替换

3. **数据库权限**
   - 需要在云开发控制台配置数据库权限
   - 建议初期使用"仅创建者可读写"

4. **错误处理**
   - 各页面的错误处理需要进一步完善
   - 建议添加全局错误捕获机制

## 💡 开发建议

### 1. 开发顺序建议
按照以下顺序开发，可以快速看到效果：
1. 先完成云数据库配置
2. 再完成情绪输入 → 云函数 → 保存数据库的流程
3. 然后完成天空页面的数据读取和展示
4. 最后完善图鉴和其他功能

### 2. 测试策略
- 每完成一个功能模块立即测试
- 使用真机测试云函数调用
- 准备测试数据（模拟多条日记）

### 3. 性能优化
- 天空页面默认只显示近7天的鸟类
- 图片使用云存储 CDN
- 合理使用 setData，避免频繁更新
