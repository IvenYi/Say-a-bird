## 关键修改记录

### v1 → v2（核心重构）

1. **P0-1 解决**：天空背景双机制矛盾 → 解耦为"实时光照层"（根据真实时间变化明暗色温）+"进化装饰层"（根据鸟类数量叠加云层/彩虹/星座等元素），两层叠加渲染
2. **P0-2 解决**：每日2次免费与树洞定位冲突 → 情绪记录改为免费不限次，付费点转移到VIP专属鸟类、完整情绪报告、装饰商城折扣、去广告等增值功能
3. **P0-3 解决**：MVP缺少鸟类密度处理 → 增加"近7天/本月全部"时间筛选，默认只显示近7天鸟类
4. 新增首次使用引导（Onboarding）场景
5. 调整用户分层：观鸟爱好者提升为P0用户（与情绪记录用户并列）
6. 新增商业化详述：VIP权益对比表、定价方案（月付12元/年付98元）
7. 新增日记编辑和删除功能
8. 新增通知与召回策略（每日提醒、周报通知、连续记录激励）
9. 补充MVP首批20种鸟类×情绪映射表（含鸟类小知识）
10. NLP API推荐腾讯云NLP，后端推荐微信云开发

### v2 → v3（细节打磨）

1. **P1-1 解决**：增加单日记录上限（免费20次/VIP 50次），温暖语气提示
2. **P1-2 解决**：新增账户注销（P1）和数据导出（P2）功能，满足合规要求
3. **P1-3 解决**：完善离线场景策略（本地缓存+占位状态+恢复后自动生成+通知）
4. 优化鸟类映射措辞：乌鸦从"沮丧/阴郁"改为"沉思/内省"，鸵鸟从"逃避/压力"改为"需要休息/暂停"
5. 新增广告形式与位置详述（Banner/激励视频/原生广告+每日5次上限）

- **通过方式**：3轮迭代达标确认（v1→v2→v3）
